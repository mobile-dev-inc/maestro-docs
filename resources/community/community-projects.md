### Community-built projects and integrations

* [VSCode Maestro Workbench](https://github.com/Mastersam07/maestro-workbench/): Visual Studio Code extension with IntelliSense, syntax highlighting, formatting, test execution, and output visualization
* [VSCode Maestro Assistant](https://github.com/agens-no/vscode-maestro-assistant): Visual Studio Code extension with Maestro shortcuts and simplified workflows for creating and running flows
* [React Native Maestro](https://github.com/kiki-le-singe/react-native-maestro): Sample project using Maestro to test a React Native app built with Expo
* [E2E testing of React Native apps using EAS and Maestro](https://docs.expo.dev/build-reference/e2e-tests/): Expo documentation
* [Codemagic CI integration](https://docs.codemagic.io/integrations/maestro-integration/): Run Maestro tests in Codemagic CI/CD pipelines
* [Fastlane Plugin](https://github.com/inf2381/fastlane-plugin-maestro): Fastlane plugin for running Maestro flows
* [ASDF Plugin](https://github.com/dotanuki-labs/asdf-maestro): ASDF version manager plugin for installing Maestro CLI
* [Maestro Test Results to Slack](https://github.com/brett-james-rocketlab/maestro-test-results-to-slack): Parses Maestro results in a folder and posts to Slack
